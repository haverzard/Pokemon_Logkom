Tokemon Pro & Log Kelompok 4

Platform : GNU Prolog (Linux Ubuntu)

Program yang digunakan dan cara install:
- GNU Prolog
Cara Install: sudo apt install gprolog
- VLC
Cara Install: sudo apt install vlc

Cara Menjalankan Game:
Pertama-tama, compile file main.pl dengan gplc dengan syntax 'gplc main.pl' pada terminal. Lalu, akan muncul file binary main yang dapat langsung dijalankan. Sebelum menjalankan file binary tersebut, pastikan folder SoundTrack (dan isinya), file song.pl, file frame.pl, dan file petabaru.txt berada pada folder yang sama dengan file binary tersebut. Jalankan program dengan syntax './main' pada terminal.

Untuk kemudahan permainan, tersedia juga loadfile bernama save.pl yang dapat diload dengan command 'loadGame(save).' dalam program main. Pastikan save.pl berada pada folder yang sama dengan file main.
